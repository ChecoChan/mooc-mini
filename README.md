# Mooc Mini
Mooc Mini 在线学习网站

#### 介绍
本项目是一个类似于 Mooc 的在线教育平台。核心模块包括：内容管理、媒资管理、课程搜索、订单支付、选课管理、认证授权等。

项目采用前后端分离的技术架构，使用 Spring Cloud 技术栈构建微服务，数据库采用 MySQL，还使用了 Nacos、Spring Cloud Gateway、Redis、RabbitMQ、Elasticsearch、XXL-Job 等。
